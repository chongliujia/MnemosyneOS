# Email Inbox

## Purpose
Read recent email messages through the configured email connector and summarize them into an artifact.

## When To Use
Use this skill when a task needs inbox inspection, recent mail triage, or a filtered mailbox check.

## Required Metadata
- `query` (optional; filters subject/from/snippet)

## Connector
- Requires `MNEMOSYNE_MAILDIR_ROOT`
- Optional `MNEMOSYNE_MAILDIR_MAILBOX`
